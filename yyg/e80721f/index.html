рҹ”ңд»Җд№ҲжҳҜAnsibleпјҹрҹ”ҡ

Ansible жҳҜдёҖж¬ҫ IT иҮӘеҠЁеҢ–е·Ҙе…·гҖӮдё»иҰҒеә”з”ЁеңәжҷҜжңүй…ҚзҪ®зі»з»ҹгҖҒиҪҜд»¶йғЁзҪІгҖҒжҢҒз»ӯеҸ‘еёғеҸҠдёҚеҒңжңҚе№іж»‘ж»ҡеҠЁжӣҙж–°зҡ„й«ҳзә§д»»еҠЎзј–жҺ’гҖӮ

Ansible жҸҗдҫӣејҖжәҗиҮӘеҠЁеҢ–пјҢеҸҜйҷҚдҪҺеӨҚжқӮжҖ§е№¶еңЁд»»дҪ•ең°ж–№иҝҗиЎҢгҖӮдҪҝз”Ё Ansible еҸҜд»ҘиҮӘеҠЁжү§иЎҢеҮ д№Һд»»дҪ•д»»еҠЎгҖӮ

жһ¶жһ„

AnsibleпјҡAnsibleж ёеҝғзЁӢеәҸгҖӮ
HostInventoryпјҡи®°еҪ•з”ұAnsibleз®ЎзҗҶзҡ„дё»жңәдҝЎжҒҜпјҢеҢ…жӢ¬з«ҜеҸЈгҖҒеҜҶз ҒгҖҒipзӯүгҖӮ
PlaybooksпјҡвҖңеү§жң¬вҖқYAMLж јејҸж–Үд»¶пјҢеӨҡдёӘд»»еҠЎе®ҡд№үеңЁдёҖдёӘж–Үд»¶дёӯпјҢе®ҡд№үдё»жңәйңҖиҰҒи°ғз”Ёе“ӘдәӣжЁЎеқ—жқҘе®ҢжҲҗзҡ„еҠҹиғҪгҖӮ
CoreModulesпјҡж ёеҝғжЁЎеқ—пјҢдё»иҰҒж“ҚдҪңжҳҜйҖҡиҝҮи°ғз”Ёж ёеҝғжЁЎеқ—жқҘе®ҢжҲҗз®ЎзҗҶд»»еҠЎгҖӮ
CustomModulesпјҡиҮӘе®ҡд№үжЁЎеқ—пјҢе®ҢжҲҗж ёеҝғжЁЎеқ—ж— жі•е®ҢжҲҗзҡ„еҠҹиғҪпјҢж”ҜжҢҒеӨҡз§ҚиҜӯиЁҖгҖӮ
ConnectionPluginsпјҡиҝһжҺҘжҸ’д»¶пјҢAnsibleе’ҢHostйҖҡдҝЎдҪҝз”Ё

AnsibleйҮҚиҰҒжҰӮеҝө

жҺ§еҲ¶иҠӮзӮ№

жҺ§еҲ¶иҠӮзӮ№жҳҜиҝҗиЎҢ Ansible Ansible CLI е·Ҙе…· зҡ„жңәеҷЁгҖӮе®ғиҙҹиҙЈдёҺеҸ—жҺ§иҠӮзӮ№пјҲиў«з®ЎзҗҶиҠӮзӮ№пјүиҝӣиЎҢйҖҡдҝЎпјҢеҸ‘еҮәе‘Ҫд»Өе№¶жү§иЎҢд»»еҠЎгҖӮ

еҸ—жҺ§иҠӮзӮ№пјҲиў«з®ЎзҗҶиҠӮзӮ№пјү

д№ҹз§°дёәвҖңдё»жңәвҖқпјҢиҝҷдәӣжҳҜз”Ё Ansible з®ЎзҗҶзҡ„зӣ®ж Үи®ҫеӨҮгҖӮAnsibleжҺ§еҲ¶иҠӮзӮ№дјҡеңЁиҝҷдәӣжңәеҷЁдёҠжү§иЎҢд»»еҠЎгҖӮеҸ—жҺ§иҠӮзӮ№дёҚйңҖиҰҒе®үиЈ…д»»дҪ•д»ЈзҗҶгҖӮ

еә“еӯҳ (Inventory)

еҢ…еҗ«иҰҒз®ЎзҗҶзҡ„зӣ®ж Үдё»жңәжҲ–иҠӮзӮ№зҡ„жё…еҚ•гҖӮе®ғжҳҜ Ansible з”ЁжқҘе®ҡд№үзӣ®ж Үдё»жңәдҝЎжҒҜзҡ„й…ҚзҪ®ж–Үд»¶пјҢж”ҜжҢҒйқҷжҖҒжҲ–еҠЁжҖҒзҡ„дё»жңәжё…еҚ•гҖӮе®ғиҝҳз”ЁдәҺеҲҶй…Қз»„пјҢиҝҷж—ўе…Ғи®ёеңЁеү§йӣҶдёӯйҖүжӢ©иҠӮзӮ№пјҢд№ҹе…Ғи®ёжү№йҮҸеҲҶй…ҚеҸҳйҮҸгҖӮ

еү§жң¬пјҲPlaybookпјү

Playbook жҳҜ Ansible й…ҚзҪ®з®ЎзҗҶзҡ„ж ёеҝғйғЁеҲҶпјҢжҳҜдёҖз»„з”ЁдәҺиҮӘеҠЁеҢ–жү§иЎҢд»»еҠЎзҡ„ YAML ж–Үд»¶гҖӮPlaybook еҸҜд»Ҙжү§иЎҢеӨҡдёӘжӯҘйӘӨпјҲдҫӢеҰӮе®үиЈ…иҪҜд»¶гҖҒеҗҜеҠЁжңҚеҠЎзӯүпјүпјҢе№¶е®ҡд№үжү§иЎҢйЎәеәҸгҖҒд»»еҠЎзӯүгҖӮ

жЁЎеқ— (Modules)

жЁЎеқ—жҳҜ Ansible дёӯжү§иЎҢе®һйҷ…д»»еҠЎзҡ„еҚ•е…ғгҖӮе®ғд»¬жҳҜй«ҳеәҰеҸҜеӨҚз”Ёзҡ„д»Јз Ғеқ—пјҢиҙҹиҙЈжү§иЎҢзү№е®ҡзҡ„ж“ҚдҪңпјҲеҰӮе®үиЈ…иҪҜд»¶еҢ…гҖҒй…ҚзҪ®ж–Үд»¶зӯүпјүгҖӮ

и§’иүІ (Roles)

и§’иүІжҳҜ Playbook зҡ„дёҖз§Қз»„з»Үж–№ејҸпјҢз”ЁдәҺжҠҠеӨҡдёӘд»»еҠЎгҖҒж–Үд»¶гҖҒжЁЎжқҝгҖҒеҸҳйҮҸзӯүз»„з»ҮжҲҗдёҖдёӘе®Ңж•ҙзҡ„еҠҹиғҪжЁЎеқ—гҖӮи§’иүІеё®еҠ©е°ҶеӨҚжқӮзҡ„й…ҚзҪ®жӢҶеҲҶжҲҗеҸҜеӨҚз”Ёзҡ„з»„д»¶гҖӮеҸҜеңЁеү§йӣҶеҶ…йғЁдҪҝз”Ёзҡ„еҸҜйҮҚз”Ё Ansible еҶ…е®№гҖӮ

д»»еҠЎ (Tasks)

д»»еҠЎжҳҜ Playbook дёӯе®ҡд№үзҡ„е…·дҪ“ж“ҚдҪңгҖӮжҜҸдёӘд»»еҠЎйҖҡеёёдјҡи°ғз”ЁдёҖдёӘ Ansible жЁЎеқ—жқҘжү§иЎҢзү№е®ҡзҡ„ж“ҚдҪңгҖӮжҳҜad-hoc жӣҙйҖӮеҗҲдёҙж—¶жү§иЎҢе‘Ҫд»Өзҡ„жү§иЎҢеңәжҷҜгҖӮ

жҸ’д»¶

жү©еұ• Ansible ж ёеҝғеҠҹиғҪзҡ„д»Јз ҒзүҮж®өгҖӮжҸ’д»¶еҸҜд»ҘжҺ§еҲ¶жӮЁеҰӮдҪ•иҝһжҺҘеҲ°иў«з®ЎзҗҶиҠӮзӮ№пјҲиҝһжҺҘжҸ’д»¶пјүгҖҒж“ҚдҪңж•°жҚ®пјҲиҝҮж»ӨеҷЁжҸ’д»¶пјүпјҢз”ҡиҮіжҺ§еҲ¶еңЁжҺ§еҲ¶еҸ°дёӯжҳҫзӨәзҡ„еҶ…е®№пјҲеӣһи°ғжҸ’д»¶пјүгҖӮ

йӣҶеҗҲ

Ansible еҶ…е®№зҡ„еҲҶеҸ‘ж јејҸпјҢеҸҜд»ҘеҢ…еҗ«еү§жң¬гҖҒи§’иүІгҖҒжЁЎеқ—е’ҢжҸ’д»¶гҖӮ

Ansibleд»»еҠЎжү§иЎҢжЁЎејҸ

Ansible зі»з»ҹз”ұжҺ§еҲ¶дё»жңәеҜ№иў«з®ЎиҠӮзӮ№зҡ„ж“ҚдҪңж–№ејҸеҸҜеҲҶдёәдёӨзұ»пјҢеҚіad-hocе’Ңplaybookпјҡ

  • ad-hocжЁЎејҸ(зӮ№еҜ№зӮ№жЁЎејҸ)
    гҖҖгҖҖдҪҝз”ЁеҚ•дёӘжЁЎеқ—пјҢж”ҜжҢҒжү№йҮҸжү§иЎҢеҚ•жқЎе‘Ҫд»ӨгҖӮad-hoc е‘Ҫд»ӨжҳҜдёҖз§ҚеҸҜд»Ҙеҝ«йҖҹиҫ“е…Ҙзҡ„е‘Ҫд»ӨпјҢиҖҢдё”дёҚйңҖиҰҒдҝқеӯҳиө·жқҘзҡ„е‘Ҫд»ӨгҖӮ
  • playbookжЁЎејҸ(еү§жң¬жЁЎејҸ)
    гҖҖгҖҖжҳҜAnsibleдё»иҰҒз®ЎзҗҶж–№ејҸпјҢд№ҹжҳҜAnsibleеҠҹиғҪејәеӨ§зҡ„е…ій”®жүҖеңЁгҖӮplaybookйҖҡиҝҮеӨҡдёӘtaskйӣҶеҗҲе®ҢжҲҗдёҖзұ»еҠҹиғҪгҖӮеҸҜд»Ҙз®ҖеҚ•ең°жҠҠplaybookзҗҶи§ЈдёәйҖҡиҝҮз»„еҗҲеӨҡжқЎad-hocж“ҚдҪңзҡ„й…ҚзҪ®ж–Үд»¶гҖӮ
