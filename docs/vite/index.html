vitepress搭建文档教程 ​

更新: Invalid Date 字数: 0 字 时长: 0 分钟

本教程是使用vitepress搭建在线教程手册，分为搭建文档和美化两部分。其中使用的vitepress版本是vitepress v2.0.0-alpha.12。

建议搭建时在VS Code中进行，相较于使用系统自带的文本编译器主要改善有:

  • 目录清晰，不用在调试的时候来回切换文件管理器
  • 编辑配置的时候，如果有错误，会立即有提示
